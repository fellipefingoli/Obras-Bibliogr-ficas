# README

## Pre requiriments

* Ruby 2.5.8
* Node 12.18+
* SQLite3

## Installment steps

If using Ruby version without `bundler` run:

```gem install bundler```

after installing bundle run:

```bundle install```

create database with:

```rails db:create```

and after that:

```rails db:migrate```

then finally run:

```rails s```
